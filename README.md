# IPTV Downloader

🎬 Python-скрипт для скачивания и записи видео из IPTV плейлистов.

## Особенности

- 📺 **Интерактивный режим** - удобный выбор каналов по номеру или названию
- 📅 **EPG поддержка** - электронная программа передач с кэшированием
- 📼 **Запись из архива** - скачивание передач из телеархива
- 🔴 **Прямой эфир** - запись текущего эфира с выбором длительности
- 🎯 **Выбор качества** - автоматическое определение лучшего качества потока
- 💾 **Кэширование** - EPG и история плейлистов сохраняются локально
- 🛡️ **Обработка ошибок** - корректная обработка сбоев и прерываний

## Требования

- Python 3.8+
- ffmpeg (должен быть установлен в системе)

## Установка

### 1. Клонирование репозитория

```bash
git clone https://github.com/yourusername/iptv-downloader.git
cd iptv-downloader
```

### 2. Установка зависимостей

```bash
pip install -r requirements.txt
```

### 3. Установка ffmpeg

**macOS:**
```bash
brew install ffmpeg
```

**Linux (Debian/Ubuntu):**
```bash
sudo apt install ffmpeg
```

**Windows:**
Скачайте с [официального сайта](https://ffmpeg.org/download.html) и добавьте в PATH.

## Использование

### Быстрый старт

```bash
python iptv_downloader.py
```

Или как модуль:
```bash
python -m iptv_downloader
```

### Рабочий процесс

1. **Введите ссылку на плейлист**
   - Скрипт запросит URL IPTV плейлиста (формат .m3u или .m3u8)
   - Поддерживаются ссылки на локальные и удалённые плейлисты

2. **Выберите канал**
   - Введите номер канала из списка
   - Или введите часть названия для поиска
   - При нескольких совпадениях выберите нужный канал

3. **Выберите режим записи**
   - **Архив** - скачивание передач из телеархива (требуется EPG)
   - **Прямой эфир** - запись текущего эфира

4. **Настройте запись**
   - Для прямого эфира: укажите длительность (в минутах или секундах)
   - Для архива: выберите передачу из списка

5. **Запись**
   - Нажмите `Ctrl+C` для досрочной остановки
   - Файл сохраняется в папку `downloads/`

## Примеры

### Поиск канала по названию

```
Введите номер канала или часть названия для поиска:

Найдено совпадений: 3
  36. ТНТ HD [Общие]
  153. ТНТ HD orig [Общие]
  1942. ТНТ HD 50 [Развлекательные]

```
Выберите режим записи:
1. Запись из архива (если доступно)
2. Запись прямого эфира
> 1

Выберите период:
1. Вчера
2. Сегодня
3. Все доступные

```

  - в минутах (например, 30)
  - в секундах с 's' (например, 90s)

> 30

[*] Запуск записи...
    Файл: ТНТ HD_20240115_143000.mp4
    Длительность: 30м 0с
```

## Конфигурация

### Настройки (src/iptv_downloader/config.py)

| Параметр | Значение | Описание |
|----------|----------|----------|
| `EPG_CACHE_HOURS` | 6 | Время кэширования EPG (часы) |
| `MAX_FILE_SIZE_GB` | 10 | Порог предупреждения о размере (ГБ) |
| `DEFAULT_TIMEOUT` | 30 | Таймаут HTTP запросов (сек) |

## Формат плейлиста

Поддерживаются плейлисты формата M3U/M3U8:

```m3u
#EXTM3U url-tvg="http://epg.example.com/guide.xml"
#EXTINF:0 tvg-id="353" tvg-logo="http://logo.png" group-title="Entertainment" tvg-rec="7",ТНТ HD
http://example.com/stream.m3u8
```

### Поддерживаемые атрибуты

- `tvg-id` - идентификатор канала для EPG
- `tvg-logo` - URL логотипа
- `group-title` - группа канала
- `tvg-rec` - доступность архива (дни)

## Troubleshooting

### ffmpeg не найден

```bash
# Проверка установки
ffmpeg -version

# Если не установлен - см. раздел Установка выше
```

### Ошибка при записи потока

- Проверьте доступность потока в VLC плеере
- Убедитесь что есть место на диске
- Проверьте логи в файле `error.log`

### EPG не загружается

- Проверьте доступность URL EPG в браузере
- Удалите файлы `history/epg*` для принудительного обновления
- Некоторые плейлисты не содержат URL EPG - используйте резервный
